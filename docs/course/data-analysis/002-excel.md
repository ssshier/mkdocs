## 常用函数

`sum`: 求和

`sumif`: 单条件求和

`sumifs`: 多条件求和

`subtotal`: 根据筛选求和

`if`: 逻辑判断

`vlookup`: 连接匹配数据

`match`: 查找数值在区域中的位置, 行/列

`index`: 根据区域的位置返回数值

`match`, `index` 一起使用: 自动根据列名查找数据

`sumifs`, `match`, `index` 顶级用法: 自动根据列名的数据和条件进行求和

::: tip
要注意取值区域是否需要改变, 不需要改变的话建议锁定
:::
